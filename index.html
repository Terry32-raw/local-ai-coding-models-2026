Plain-English takeaway

  • If you have 16GB RAM, stay conservative.
  • If you have 32GB RAM, Qwen3.6‑27B becomes realistic.
  • If you have 64GB+, try Qwen3‑Coder‑Next.
  • Huge frontier models are cool, but not “normal computer” local.

The cleaned-up list

Ranked for practical local coding use, not just leaderboard bragging rights.

1

Qwen3.6‑27B

Best practical local pick

The best balance of real coding ability and actual local runnability. Quantized GGUF builds can fit into consumer machines, and there is strong community evidence of useful day-to-day coding work.

Hardware32GB ideal; 16GB tight
Best forRepo help, scripts, app edits
Run withllama.cpp / LM Studio
Reality checkLower context on small Macs
2

Qwen3‑Coder‑Next

Best coding-agent candidate

A coding-focused MoE model from the Qwen family, aimed at agentic workflows and local development. It is a stronger target if your computer has enough memory headroom.

Hardware64GB+ preferred
Best forAgent loops, tool use
LicenseApache 2.0 reported
Reality checkToo big for casual 16GB use
3

Devstral Small

Safest modest-machine choice

Mistral and All Hands built Devstral for agentic software engineering. It is not the newest hype model, but it remains one of the most practical open coding models for local tool loops.

Hardware32GB good; 16GB possible
Best forReliable coding assistant
LicenseApache 2.0
Reported score53.6% SWE‑Bench Verified
4

Cohere North Mini Code

Newest small MoE to watch

A June 2026 developer-focused MoE model: 30B total / 3B active, designed for terminal-based agentic coding. Promising, Apache 2.0, but still a “test before trusting” pick.

Hardware32GB+ with quant
Best forExperimenting with new agent models
WeightsBF16 / FP8 on Hugging Face
Reality checkLess proven than Qwen/Devstral
5

GLM‑5.2

Frontier model, not normal-local

Very impressive current open model: MIT licensed, 1M context, strong long-horizon coding claims. But it is frontier-scale — better for hosted or serious workstation use than a normal local computer.

HardwareWorkstation / server
Best forLong-horizon agentic work

By machine size

Use this if you just want to know what to try first.

16GB Mac / PC

  • Try Devstral Small low-quant first.
  • Try Qwen3.6‑27B only with lower quant and lower context.
  • Avoid giant MoE frontier models locally.

32GB Mac / PC

  • Start with Qwen3.6‑27B Q4.
  • Keep Devstral Small as the steady fallback.
  • Test North Mini Code if good local packaging exists.

64GB+ / strong GPU

  • Try Qwen3‑Coder‑Next.
  • Compare against Qwen3.6‑27B for speed/quality.
  • Only experiment with GLM‑5.2 if you enjoy infrastructure pain.
